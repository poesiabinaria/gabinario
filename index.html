1. O Presente Diário

Abrir Projeto

O Presente Diário é um projeto pessoal que traz para o usuário a oportunidade de experimentar o passado, presente e futuro assim como eles são. Nele, não há a possibilidade de editar as folhas futuramente, apenas de recordar o passado e de escrever o presente assim como ele é. Foi meu primeiro projeto mais complexo, onde ganhei conhecimentos de Web Design, UI/UX (e, também, das tecnologias abaixo).

Nele, é utilizado o famoso trio do front-end (HTML, CSS e JavaScript) e Node.js com MongoDB no back-end — que implementei especialmente para deixar o front-end interativo. Você pode criar uma conta ou usar essa para se logar:

2. Calculadora para Diluição de Óleos Essencais

Abrir Projeto

Pequeno projeto pessoal para uma comunidade de aromaterapia no Facebook: uma calculadora de diluição de óleos essenciais em óleos vegetais. O cálculo, se feito a mão, pode ser trabalhoso porque envolve duas regras de três, porcentagem e relação entre quantidade de gotas e mL. Digitar só duas informações e mandar pro JavaScript é bem mais fácil!

Durante seu desenvolvimento, consegui melhorar e solidificar alguns conhecimentos em JavaScript e CSS.
